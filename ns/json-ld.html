The JSON-LD Vocabulary

This is a vocabulary document and is used to achieve certain features of the JSON-LD language.

Vocabulary Terms

The vocabulary terms below constitute the complete JSON-LD vocabulary.

JSON-LD context

http://www.w3.org/ns/json-ld#context

This link relation is used to reference a JSON-LD context from a JSON document so that it can be interpreted as JSON-LD.

You can read more about this feature in the JSON-LD Syntax specification.

Expanded JSON-LD document form

http://www.w3.org/ns/json-ld#expanded

This profile URI is used to request or specify expanded JSON-LD document form.

You can read more about this feature in the JSON-LD Syntax specification.

Compacted JSON-LD document form

http://www.w3.org/ns/json-ld#compacted

This profile URI is used to request or specify compacted JSON-LD document form.

You can read more about this feature in the JSON-LD Syntax specification.

Flattened JSON-LD document form

http://www.w3.org/ns/json-ld#flattened

This profile URI is used to request or specify flattened JSON-LD document form.

You can read more about this feature in the JSON-LD Syntax specification.
